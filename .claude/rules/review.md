# Qiita記事 レビュールール

## 必須参照
- `CONTRIBUTING_QIITA.md` — 全チェック項目の基準
- `context/knowledge/review-lessons.md` — 過去のレビュー教訓

## チェック項目
- フロントマターの完全性（title, tags, private）
- 見出し階層の正しさ（H2 → H3 → H4、H1不使用）
- コードブロックの言語指定
- 技術的正確性（API使用法、バージョン情報）
- 誤字脱字・表記揺れ
- ですます調の統一
- 文字数（本文2,000〜6,000文字目安）
